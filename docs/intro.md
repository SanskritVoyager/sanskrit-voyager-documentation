# Documentation Intro

Welcome to **Sanskrit Voyager**, a web application to explore the *Sanskrit* literature.

The goal of the application is to make the original Sanskrit texts accessible regardless of the knowledge of the language.

### Overview

Navigate through the documentation sections to learn more:
