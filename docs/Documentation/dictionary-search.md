The purpose of the dictionary search is to allow anyone to search words exactly as they appear in the text, with no changes. 

To this purpose, the website offers a transliteration free, inflected and sandhi-ed multi dictionary search. 

1. Text can be searched in any transliteration format. Exact matches can also be written without diacritics (samskara matches **saṃskāra).** 
2. Inflected and sandhi-ed forms are accepted as text input. (’*eṣyāmi’ is matched as ‘i’*)
3. Compounds can be used as input as well (i.E: ‘vitarkavicārānandāsmitārūpānugamāt’). You can simply scroll down for all the words matched. 
4. The website tries to recover for each word the components, the grammatical type, and the case for nouns or inflection for verbs. 
5. Clicking on the sheet button, it is possible to have the inflection table for the word, with the case of the original word highlighted. 
6. The dictionary entry for the selected dictionaries is provided. Default is Monnier Williams. If the selected word is not in one of the chosen dictionaries, it automatically selects a dictionary that has it, if available. 
7. All the names and Sanskrit terms inside the dictionary voices are clickable. On click it sends to the entry for the names. The references are provided with no abbreviation.
