This is the most ambitious feature on the website. 

You can access the feature by clicking on "Corpus Search" on the navbar. 
It is currently only displaying correctly only on desktop. 

There is a major upcoming upgrade to make it display better on desktop and correctly on any device coming. 

Using the full corpus search, you can search terms or passages in the whole Sanskrit literature. 

Grouped by book groups by the results on the book. Clicking on the books scrolls to the passages and shows the matches on the scrollbar. Click on the scrollbar ticks to go to the passages. 

You can use filters to search in specific content. Currently only include works.
