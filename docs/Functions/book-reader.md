Building on the dictionary search and text analysis tool, the website provides an interactive book reader. At the moment we have about 800 text, all coming from Gretil. In the next update the number will go up to more than 1600, incorporating multiple sources. The database is ready, the main problem is the formatting, as there are always inconsistent tags. 

1. Selecting in the Book to import menu a text, books can be imported. Books can be searched by text name or author. The search is case and diacritics insensitive. 
2. Once a book is selected, it is displayed on the center of the screen. Metadata for the books can be accessed clicking on the metadata dropdown. 
3. All the book is interactive, and each word can be clicked to access the analysis for the terms. 
4. Some of the books have been machine translated experimentally. The quality and worth of those translation is yet to be assessed. 
5. For the translated books, the segment control on the left allows access to ‘only translation’, ‘translation and original’ and ‘only original’. 
6. The Gretil corpus has been converted from XML to JSON for better rendering. Unfortunately There was no standard for the XML, which have to be manually decoded and understood.
